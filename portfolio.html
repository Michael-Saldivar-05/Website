Python project with Mohnish!

Link to our Safari Park

My first project in APCSP 2022-23 was to create a small interactive safari park drawing using the python turtle module.

The project first creates a truck on a safari background. It then prompts the user to choose a direction.

Depending on the direction chosen, an image of a lion or elephant will be displayed.

This project uses the replit console so a fork (separate copy) is necessary for running the prompts.

PLTW Data Analysis Project



The PLTW assignment had us interpreting data from a space rover.

Data was given in a very compressed format and had to be both visualized and sorted.

Using the data we deciphered, we found that the rover was located on a desert plain.

This conclusion came about due to the constant loud sounds and varying regularity of winds that were only from desert plains.


The result turned out to be incorrect as we mixed up temperature and light. We also interpreted the gyro graph incorrectly.
